DecodeX does not store any user data on its servers. All outputs generated from user input are stored locally on the user’s physical device. If you delete the app, you also delete all the data associated with it.

5. User Responsibility

Since DecodeX does not retain or process user data outside the context of third-party services, we do not have any obligation to secure or protect user data beyond what is outlined by these services. Users are encouraged to back up their results manually if required, as deleting the app will result in permanent data loss.

6. Security

DecodeX does not handle or store user data, and thus, we have no security obligations regarding user data. Any security concerns should be directed to the third-party services mentioned above.

7. User Rights

As DecodeX does not retain user data, we cannot offer mechanisms for accessing, modifying, or deleting user data. All interactions with the app occur in real-time and are not stored post-processing.

8. Compliance with Data Protection Laws

DecodeX processes all data anonymously and does not retain any identifiable user information. However, certain regions may have data protection laws (e.g., GDPR, CCPA) that impose additional responsibilities. Since DecodeX relies on third-party tools, their compliance policies apply. If you have concerns about compliance, we recommend contacting the third-party services directly.
